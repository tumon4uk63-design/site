# Сайт‑витрина для АО «Атомдата‑Интеграция»

Одностраничный сайт по продаже оборудования и ИТ‑услуг для компании «Атомдата‑Интеграция», основанный на информации с официального сайта `https://atomdata-int.ru/`.

## Структура

- `index.html` — основная страница (герой‑блок, решения, каталог оборудования, услуги, преимущества, контакты и формы заявок).
- `styles.css` — оформление в современном тёмном стиле.
- `script.js` — интерактив: фильтрация каталога, модальное окно, плавная прокрутка, демонстрационная обработка форм.

## Как открыть

1. Откройте папку `atomdata-site` на рабочем столе.
2. Дважды щёлкните по файлу `index.html` — он откроется в браузере.

Для боевого использования подключите обработку форм на backend и при необходимости интеграцию с CRM/почтой.
